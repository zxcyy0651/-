# HELLO ERICA
SF 감성 OTT + 캐릭터 성장 시스템 웹 페이지

## 📌 프로젝트 소개
본 프로젝트는 SF 테마의 OTT 메인 화면을 웹으로 구현한 UI 프로젝트입니다.  
사용자는 메인 히어로 화면과 콘텐츠 카드 UI를 통해 OTT 플랫폼의 구조를 시각적으로 경험할 수 있습니다.  
향후 로그인, 마이페이지, 캐릭터 성장 시스템과 연계 가능한 확장 구조로 설계되었습니다.

## 🔧 사용 기술
- HTML5
- CSS3
- Git / GitHub
- GitHub Pages

## ✅ 구현 기능
- 메인 히어로 섹션 UI
- 상단 네비게이션 바
- OTT 스타일 콘텐츠 카드 UI
- Git 기반 버전 관리
- GitHub Pages를 통한 웹 배포
