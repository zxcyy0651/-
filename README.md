# 🎬 HYUFLIX

SF 감성 OTT 웹 서비스 (Open Source SW 활용 프로젝트)

## 1. 프로젝트 개요

HYUFLIX는 Netflix UI/UX를 벤치마킹하여 제작한 **SF 테마 기반 OTT 웹 서비스**이다.
HTML, CSS, JavaScript를 활용하여 **정적인 웹이 아닌 동적으로 콘텐츠가 생성·필터링되는 구조**로 구현하였다.
GitHub와 GitHub Pages를 활용하여 **버전 관리 및 웹 배포**까지 완료한 실사용 가능한 프로젝트이다.

---

## 2. 프로젝트 목표

* OTT 서비스의 기본 UI/UX 구조 구현


## 5. 주요 기능

### ✅ 1) 메인 영화 목록 자동 출력

* JavaScript 배열(`movies`)을 기반으로 영화 카드가 **자동 생성**
* 기존 하드코딩 방식이 아닌 **동적 렌더링 방식 적용**

### ✅ 2) 5×4 그리드 포스터 배치

* CSS Grid를 활용하여 **5열 고정 배치**
* 화면 크기에 따라 자동 반응형 처리

### ✅ 3) 마우스 오버 정보 표시

* 영화 카드에 마우스를 올리면:

  * 영화 제목
  * 평점(5점 만점)
    가 자동 표시됨

### ✅ 4) 영화 상세 정보 모달

* 영화 클릭 시:

  * 포스터 이미지
  * 영화 제목
  * 줄거리
  * 평점
    이 **모달(팝업) 형태로 출력**

### ✅ 5) 영화 / 시리즈 필터링

* 상단 메뉴에서:

  * **홈** → 전체 콘텐츠 출력
  * **영화** → 영화 타입만 출력
  * **시리즈** → 시리즈 타입만 출력
* JavaScript `filter()`를 활용한 **동적 데이터 필터링 구현**

### ✅ 6) GitHub Pages 웹 배포

* GitHub 저장소와 연동하여 **실제 웹 주소로 배포 완료**
* Git을 이용한 코드 변경 이력 관리

## 7. 구현 방식 설명

### ▶ 동적 렌더링 방식

```js
movies.forEach(m => {
  const card = document.createElement("div");
  card.style.backgroundImage = `url(${m.img})`;
  grid.appendChild(card);
});
```

* HTML에 이미지 태그를 직접 나열하지 않고
* **JS가 자동으로 UI를 생성하는 구조**로 구현

### ▶ 필터링 구조

```js
movies.filter(m => m.type === "movie")
```

* 배열 필터링을 이용해 **실시간 콘텐츠 변경**
