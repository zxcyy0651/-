# 🎬 HYUFLIX

**SF 감성 OTT 스트리밍 플랫폼**

넷플릭스 스타일의 영화 추천 웹사이트 + 게임 시스템

[🔗 Live Demo](https://zxcyy0651.github.io/-/)

---

## ✨ 주요 기능

### 🎮 게임 시스템
- **마일리지 시스템**: 영화 시청, 예고편 재생, 찜하기로 마일리지 획득
- **레벨 시스템**: XP를 쌓아 레벨업하고 보상 획득
- **상점**: 마일리지로 SF 테마 아이템 구매
- **캐릭터 꾸미기**: 구매한 아이템으로 나만의 캐릭터 커스터마이징
- **일일 미션**: 매일 새로운 미션으로 추가 보상
- **업적 시스템**: 다양한 업적 달성으로 특별 보상

### 🎥 OTT 기능
- **영화/시리즈 필터링**: 장르별, 타입별 분류
- **검색 기능**: 제목, 장르, 설명으로 검색
- **예고편 재생**: YouTube 영상 통합
- **찜하기**: 좋아하는 콘텐츠 저장
- **평점 시스템**: 각 콘텐츠 평점 표시

### 🎨 디자인
- **사이버펑크 UI**: SF 감성의 네온 그라데이션
- **반응형 디자인**: 모바일/태블릿/데스크톱 지원
- **인터랙티브 애니메이션**: 부드러운 호버 효과 및 트랜지션

## 🛠️ 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: Flexbox, Grid, 애니메이션
- **JavaScript (ES6+)**: 바닐라 JS, LocalStorage
- **GitHub Pages**: 무료 호스팅
